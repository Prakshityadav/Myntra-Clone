# Myntra Clone

A responsive e-commerce web application inspired by Myntra, built using ReactJS. The project replicates key online shopping functionalities including product browsing, category filtering, cart management, and responsive user interface design.

## Features

* Product listing and browsing
* Category-based filtering
* Add to Cart functionality
* Cart item management
* Responsive UI for desktop and mobile devices
* Dynamic rendering using React Components
* State management using React Hooks

## Tech Stack

### Frontend

* ReactJS
* JavaScript (ES6+)
* HTML5
* CSS3

### Tools

* Git
* GitHub
* Vite

## Installation

1. Clone the repository

```bash
git clone https://github.com/Prakshityadav/Myntra-Clone.git
```

2. Navigate to project directory

```bash
cd Myntra-Clone
```

3. Install dependencies

```bash
npm install
```

4. Run the application

```bash
npm run dev
```
